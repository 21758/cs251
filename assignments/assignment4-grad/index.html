Downloaded Image Viewer (Assignment 4)

Overview of the Downloaded Image Viewer Assignment

The purpose of this assignment is to give you experience using Android to extend your assignment 3 solution by developing an app containing a main Activity that prompts the user for a URL to an image and then uses Intents, a DownloadImageService, and the Gallery Activity to download the image and view it via the Gallery app.

Undergraduate students can use the Android IntentService to implement the DownloadImageService, which is covered in this video.

Graduate students (or students who are taking this course for graduate credit) must implement the DownloadImageService via an Android bound Service, which is covered in this video, and the Android Messenger class, which is covered in the following videos:

  • Service to Activity Communication via Android Messenger.
  • Programming Bound Services with Messengers (Part 1)
  • Programming Bound Services with Messengers (Part 2)

Skeleton Code

Skeleton code for the ugrad version of this assignment is available here. Likewise, skeleton code for the grad version of this assignment is available here. Use your GitLab account to pull the appropriate skeleton code in your repository, read it carefully, and complete the "TODO" markers.

Note that there are no automated unit tests for this assignment. Instead, you'll know the program is working when you can successfully download and view an image.

Concluding Remarks

This assignment is designed to further your understanding with programming Android in Java. It's somewhat harder than the first Android assignment since it involves background processing with an Android Service. Therefore, start early and come to office hours of you have questions or run into problems.
