Getting Started

Before you start with Assignment 1 you'll need to do the following things:
  • Download and install Eclipse Luna for Android. Instructions for doing this are available here.

  • Setup an account on GitLab, which is explained in a video by Mike Walker.

Overview of the Array Assignment

The purpose of this assignment is to give you experience using Eclipse to develop a simple Java collection.
